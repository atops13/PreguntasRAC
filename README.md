# Test Interactivo RAC

Este es un simulador de test interactivo para la asignatura de **Diseño y Gestión de Redes (RAC)**. El proyecto ha sido modernizado para ofrecer una interfaz visual premium, dinámica y con soporte para modo oscuro.

## Características Principales

- **+100 Preguntas**: Incluye una recopilación de preguntas filtradas y nuevas basadas en el temario de RAC (VLANs, Routing, OSPF, NAT, WAN, Seguridad, etc.).
- **Modo Oscuro/Claro**: Interfaz conmutable con persistencia en el navegador (Local Storage).
- **Diseño Premium**: Uso de gradientes modernos, efectos de desenfoque (glassmorphism), tipografía Inter y animaciones fluidas.
- **Feedback Inmediato**: Indica si la respuesta es correcta o incorrecta al instante, mostrando el razonamiento o la opción válida.
- **Estadísticas en Tiempo Real**: Seguimiento de la puntuación y progreso durante el test.
- **Resultados Detallados**: Resumen final con porcentaje de acierto y mensajes personalizados.

## Cómo Usar

Simplemente abre el archivo `test_rac.html` en cualquier navegador web moderno. No requiere conexión a internet ni dependencias externas pesadas (usa CSS vanilla y JS nativo).

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica.
- **CSS3**: Diseño moderno con variables CSS, Flexbox, Grid y animaciones.
- **JavaScript**: Lógica del cuestionario, barajado de preguntas y gestión de estados.
